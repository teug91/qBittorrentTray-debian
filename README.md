# qBittorrentTray

Debian version of [qBittorrentTray](https://github.com/teug91/qBittorrentTray).

## Environment

### Setup
```bash
sudo apt install python3-virtualenv
virtualenv env
```

### Activate

```bash
source env/bin/activate
```

### Install
```bash
pip install -r requirements.txt
```

### Deactivate

```bash
deactivate
```

## Build

```bash
./build.py
```
